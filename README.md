## Installation

    $ cd ~/.emacs.d/vendor
    $ git clone git://github.com/julienXX/erc-terminal-notifier.el.git

In your Emacs config:

    (add-to-list 'load-path "~/.emacs.d/vendor/erc-terminal-notifier.el")
    (require erc-terminal-notifier)
